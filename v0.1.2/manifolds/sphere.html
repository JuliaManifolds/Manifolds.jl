Sphere

Manifolds.Sphere — Type
Sphere{N} <: AbstractEmbeddedManifold{DefaultEmbeddingType}

The unit sphere manifold $𝕊^n$ represented by $n+1$-Tuples, i.e. in by vectors in $ℝ^{n+1}$ of unit length. A sphere is an embedded manifold and inherits the inner product of its embedding $\mathbb R^{n+1}$. The tangent space at point p is given by

\[T_p𝕊^n := \bigl\{ X ∈ ℝ^{n+1} : ⟨p,X⟩ = 0 \bigr \},\]

where $⟨\cdot,\cdot⟩$ denotes the inner product in the embedding $\mathbb R^{n+1}$.

This manifold is modeled as an embedded manifold to the Euclidean, i.e. several functions like the inner product and the zero_tangent_vector are inherited from the embedding.

Sphere(n)

Generate $𝕊^{n} ⊂ ℝ^{n+1}$.

Base.exp — Method
exp(M::Sphere, p, X)

Compute the exponential map from p in the tangent direction X on the Sphere M by following the great arc eminating from p in direction X.

\[\exp_p X = \cos(\lVert X \rVert_p)p + \sin(\lVert X \rVert_p)\frac{X}{\lVert X \rVert_p}X,\]

where $\lVert X \rVert_p$ is the norm on the Sphere M.

Base.log — Method
log(M::Sphere, p, q)

Compute the logarithmic map on the Sphere M, i.e. the tangent vector, whose geodesic starting from p reaches q after time 1. The formula reads for $x ≠ -y$

\[\log_p q = d_{𝕊^n}(p,q) \frac{q-⟨p,q⟩ p}{\lVert q-⟨p,q⟩ p \rVert_2},\]

and a deterministic choice from the set of tangent vectors is returned if $x=-y$, i.e. for opposite points.

ManifoldsBase.check_tangent_vector — Method
check_tangent_vector(M, p, X; check_base_point = true, kwargs... )

Check whether X is a tangent vector to p on the Sphere M, i.e. after check_manifold_point(M,p), X has to be of same dimension as p and orthogonal to p. The optional parameter check_base_point indicates, whether to call check_manifold_point for p. The tolerance for the last test can be set using the kwargs....

ManifoldsBase.distance — Method
distance(M::Sphere, p, q)

Compute the geodesic distance betweeen p and q on the Sphere M. The formula is given by the (shorter) great arc length on the (or a) great circle both p and q lie on.

\[d_{𝕊^n}(p,q) = \arccos(⟨p,q⟩).\]
ManifoldsBase.get_coordinates — Method
get_coordinates(M::Sphere, p, X, B::DefaultOrthonormalBasis)

Represent the tangent vector X at point p from the Sphere M in an orthonormal basis by rotating the vector X using the rotation matrix $2\frac{q q^\mathrm{T}}{q^\mathrm{T} q} - I$ where $q = p + (1, 0, …, 0)$.

ManifoldsBase.injectivity_radius — Method
injectivity_radius(M::Sphere[, p])

Return the injectivity radius for the Sphere M, which is globally $π$.

injectivity_radius(M::Sphere, x, ::ProjectionRetraction)

Return the injectivity radius for the ProjectionRetraction on the Sphere, which is globally $\frac{π}{2}$.

ManifoldsBase.inverse_retract — Method
inverse_retract(M::Sphere, p, q, ::ProjectionInverseRetraction)

Compute the inverse of the projection based retraction on the Sphere, i.e. rearranging $p+X = q\lVert p+X\rVert_2$ yields since $⟨p,X⟩ = 0$ and when $d_{𝕊^2}(p,q) ≤ \frac{π}{2}$ that

\[\operatorname{retr}_p^{-1}(q) = \frac{q}{⟨p, q⟩} - p.\]
ManifoldsBase.manifold_dimension — Method
manifold_dimension(M::Sphere)

Return the dimension of the Sphere(n)M`, i.e. $𝕊^n$, which is $\dim(𝕊^n) = n$.

ManifoldsBase.project — Method
project(M::Sphere, p, X)

Project the point X onto the tangent space at p on the Sphere M.

\[\operatorname{proj}_{p}(X) = X - ⟨p, X⟩p\]
ManifoldsBase.project — Method
project(M::Sphere, p)

Project the point p from the embedding onto the Sphere M.

\[\operatorname{proj}_{𝕊^n}(p) = \frac{p}{\lVert p \rVert_2}.\]
ManifoldsBase.representation_size — Method
representation_size(M::Sphere)

Return the size points on the Sphere M are represented as, i.e. for the n-dimensional Sphere it is vectors of size (n+1,).

ManifoldsBase.retract — Method
retract(M::Sphere, p, X, ::ProjectionRetraction)

Compute the retraction that is based on projection, i.e.

\[\operatorname{retr}_p(X) = \frac{p+X}{\lVert p+X \rVert_2}\]
ManifoldsBase.vector_transport_to — Method
vector_transport_to(M, p, X, q, ::ParallelTransport)

Compute the paralllel transport of the tangent vector X at p to q, provided, the geodesic between p and q is unique

\[P_{p←q}(X) = X - \frac{\langle \log_p q,X\rangle_p}{d^2_{𝕊^n}(p,q)} \bigl(\log_xy + \log_yx \bigr).\]
