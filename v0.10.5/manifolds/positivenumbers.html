Positive Numbers

The manifold PositiveNumbers represents positive numbers with hyperbolic geometry. Additionally, there are also short forms for its corresponding PowerManifolds, i.e. PositiveVectors, PositiveMatrices, and PositiveArrays.

The hyperbolic manifold of positive numbers $H^1$ is a the hyperbolic manifold represented by just positive numbers.

ManifoldsBase.change_metric ‚Äî Method
change_metric(M::PositiveNumbers, E::EuclideanMetric, p, X)

Given a tangent vector $X ‚àà T_p\mathcal M$ representing a linear function with respect to the EuclideanMetric g_E, this function changes the representer into the one with respect to the positivity metric of PositiveNumbers M.

For all $Z,Y$ we are looking for the function $c$ on the tangent space at $p$ such that

\[ ‚ü®Z,Y‚ü© = XY = \frac{c(Z)c(Y)}{p^2} = g_p(c(Y),c(Z))\]

and hence $C(X) = pX$.

ManifoldsBase.change_representer ‚Äî Method
change_representer(M::PositiveNumbers, E::EuclideanMetric, p, X)

Given a tangent vector $X ‚àà T_p\mathcal M$ representing a linear function with respect to the EuclideanMetric g_E, this function changes the representer into the one with respect to the positivity metric representation of PositiveNumbers M.

For all tangent vectors $Y$ the result $Z$ has to fulfill

\[ ‚ü®X,Y‚ü© = XY = \frac{ZY}{p^2} = g_p(YZ)\]

and hence $Z = p^2X$
